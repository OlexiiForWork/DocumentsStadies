!!!!!!!!!Установка с архивов 

1) Скачиваем с помощью команды wget
  wget <link>
2) распаковываем с помощью команды tar
  примеры
  tar xvjf <имя файла для разпаковки>
      ||||
      |||- локацыя архива  
      ||-- bzip2 алгоритм сжатия/распаковки  (-z, -j, -J, --lzma  Compress archive with gzip/bzip2/xz/lzma)
      |--- полный вывод инфы по роспаковке
      ---- Распаковать (  -c Create  -r Add/Replace  -t List  -u Update  -x Extract )
     
  tar xvjf <имя файла для разпаковки> -С <куда распаковать>

Читаем файл readme.txt инструкцыю по установке

3) Сборка и установка 
!!!!!!!!! только если в проекте есть файлы MakeFile.txt - в котором описаны инструкции для команды "make"

    3.1 [Подготовка к сборке] выполняем команду "./configure" в папке проэкта
        ./configure

    3.2 [Сборка бинарников] выполняем команду "make" в папке проэкта для сборки файлов проэкта
        make

    3.3 [Установка] выполняем команду "sudo make install" в папке проэкта для переноса файлов в коректные места
        sudo make install

    3.4 [Удаление] выполняем команду "sudo make uninstall" в папке проэкта
        !!!!!! только если не удаляли файлы/директорию с которой и выполняли установку 
        sudo make uninstall
