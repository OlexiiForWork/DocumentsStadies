## Команды Terminal !!!!!!!!!!!!!!!!!!!!!!!
        Базовые команды для управления 
----------------------------------------------------------------------------------
|           КОМАНДЫ                 |                ОПИСАНИЕ                    |
----------------------------------------------------------------------------------
|  mkdir <имя дериктории>           |        Создать директорию                  |
----------------------------------------------------------------------------------
|  ls                               |         Вывести содержымое директории      |
----------------------------------------------------------------------------------
|  ls -l                            |  Вывести содержымое директории в стролб....|
----------------------------------------------------------------------------------
|  cd ..                            |  Перейти в деректорию на уровень выше      |
----------------------------------------------------------------------------------
|  cd <путь для перехода>           |  Перейти в директрорию указаную в путь     |
----------------------------------------------------------------------------------
|  touch <имя файла>                |  Один вариант для создания файла           |
----------------------------------------------------------------------------------
|  pwd                              |  Получение текущего положения указат.      |
----------------------------------------------------------------------------------
|  cp <имя файла> <имя нового файла>|  копирование файла                         |
----------------------------------------------------------------------------------
|  rm                               |  Удаление файла                            |
----------------------------------------------------------------------------------
|  rm -rf <имя для удаления папки>  | [Опасная команда] Рекурсивное удаление дир.|
----------------------------------------------------------------------------------
|  rmdir                            | Удаление директории                        |
----------------------------------------------------------------------------------
|  man                              | Справка по команде                         |
----------------------------------------------------------------------------------
|  grep -r <текст> <имя файла>      |                                            |
----------------------------------------------------------------------------------
| tar -xvf file.tar -C /tmp/my_art/ | Чтобы извлечь файлы из file.tar в          |
|                                   |        /tmp/my_art                         |
----------------------------------------------------------------------------------



 

Чтобы вывести содержимое файла в терминале:
|
----cat <myfile>

Если файл очень большой, то можно вывести только несколько последних строк файла. 
Для этого используется команда tail:
|
----tail <mybigfile>

Для вывода содержимого файла существует команда
|
----less <mygoodfile>
    Данная команда открывает файл в режиме просмотра 
    — вы можете просматривать содержимое, нажимая клавиши Вверх/Вниз или Enter. 
    Чтобы выйти из режима просмотра нажмите клавишу q.
