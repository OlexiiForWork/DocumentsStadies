## Команды Terminal !!!!!!!!!!!!!!!!!!!!!!!

Чтобы вывести содержимое файла в терминале:
|
----cat <myfile>

Если файл очень большой, то можно вывести только несколько последних строк файла. 
Для этого используется команда tail:
|
----tail <mybigfile>

Для вывода содержимого файла существует команда
|
----less <mygoodfile>
    Данная команда открывает файл в режиме просмотра 
    — вы можете просматривать содержимое, нажимая клавиши Вверх/Вниз или Enter. 
    Чтобы выйти из режима просмотра нажмите клавишу q.
